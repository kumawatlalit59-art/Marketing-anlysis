# Marketing Trends Dashboard

This project focuses on analyzing marketing and sales performance through an interactive dashboard built using Python, SQL Server, and Power BI. The objective was to transform raw marketing data into meaningful insights that can support business decision-making.

The workflow began with data processing in Python using Pandas, followed by storing and querying the data in SQL Server. The processed data was then visualized in Power BI to create an interactive dashboard showcasing key business metrics and trends.

The dashboard provides insights into monthly sales performance, product-wise contribution, customer trends, and marketing channel effectiveness. Through various visualizations and filters, users can easily explore the data and identify top-performing products, channels, and sales patterns.

### Technologies Used

* Python (Pandas)
* SQL Server
* Power BI

### Key Features

* Monthly sales trend analysis
* Channel-wise performance tracking
* Product contribution analysis
* Customer analytics
* Interactive visualizations and filters

### Project Workflow

CSV Dataset → Python Processing → SQL Server → Power BI Dashboard

This project demonstrates skills in data cleaning, SQL querying, data visualization, and business intelligence reporting.
